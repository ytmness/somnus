# Comisiones Clip - Modelo de Cobro

## Flujo de pago

- **El cliente paga**: precio del boleto + comisión Clip (3.9% + IVA 16% sobre la comisión).
- El cargo por servicio se suma al subtotal y lo paga el cliente.

## Ejemplo: boleto de $10 MXN

| Concepto                | Monto    |
|-------------------------|----------|
| Subtotal (boleto)       | $10.00   |
| Comisión Clip (3.9%)    | $0.39    |
| IVA 16% sobre comisión  | $0.06    |
| **Total comisión**      | ≈ $0.45  |
| **Paga el cliente**     | **$10.45** |

## Ejemplo: cobro de $1,000 MXN

| Concepto                      | Monto     |
|-------------------------------|-----------|
| Subtotal                      | $1,000.00 |
| Comisión Clip (3.9%)          | $39.00    |
| IVA 16% sobre la comisión     | ≈ $6.24   |
| **Total comisión**            | ≈ $45.24  |
| **Paga el cliente**           | **$1,045.24** |
| **Recibes (neto)**            | ≈ $1,000  |
