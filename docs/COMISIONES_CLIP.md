# Comisiones Clip - Modelo de Cobro

## Flujo de pago

- **El cliente paga**: el precio del boleto (sin IVA adicional).
- **IVA no se cobra al cliente** sobre el ticket.
- **IV 16% aplica solo** a la comisión que Clip cobra al comercio.

## Ejemplo: cobro de $1,000 MXN

| Concepto                      | Monto   |
|------------------------------|---------|
| Cobro al cliente             | $1,000  |
| Comisión Clip (3.9%)         | $39     |
| IVA 16% sobre la comisión    | ≈ $6.24 |
| **Total comisión**           | ≈ $45.24|
| **Recibes (neto)**           | ≈ $954.76 |

La comisión se calcula sobre el monto total cobrado al cliente.
